### baseurl = https://api.tricked.pro/images

<br></br>
api paths:

---

<br></br>
`/upload`  
upload a image this requires a image to be in the formdata
<br></br>

---

<br></br>
`/download/:user/:key`  
download a image
<br></br>

---

<br></br>
`/delete/:user/:key/:image`  
delete a image image has to be the vanity
<br></br>

---

<br></br>
`/view/image`  
sends the image
<br></br>

---

<br></br>
`/view/image/stats`  
get some information about the image
<br></br>
